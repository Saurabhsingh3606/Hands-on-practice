### 3. E-commerce Search Function

**Objective:**
Develop a product search system using Linear Search and Binary Search algorithms and compare their performance.

**Concepts Covered:**

* Linear Search
* Binary Search
* Sorting
* Big O Notation
* Time Complexity Analysis

**Algorithms Implemented:**

* Linear Search
* Binary Search

**Complexity Analysis:**

| Algorithm     | Best Case | Average Case | Worst Case |
| ------------- | --------- | ------------ | ---------- |
| Linear Search | O(1)      | O(n)         | O(n)       |
| Binary Search | O(1)      | O(log n)     | O(log n)   |

**Conclusion:**
Binary Search provides better performance for large datasets when the data is sorted, whereas Linear Search is suitable for small or unsorted datasets.

## Big O Notation

Big O Notation is used to measure the efficiency of an algorithm as the input size increases.

Common complexities include:

* O(1) – Constant Time
* O(log n) – Logarithmic Time
* O(n) – Linear Time
* O(n log n) – Linearithmic Time
* O(n²) – Quadratic Time

Big O analysis helps in selecting the most efficient algorithm for solving a problem.
